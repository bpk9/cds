# Coinbase Design System - Lottie Files

## Get Started

Install package with `yarn add @coinbase/cds-lottie-files`

- `import { tradeStatus } from '@coinbase/cds-lottie-files/tradeStatus'` for mobile to prevent bundling all assets on build
- `import { tradeStatus } from '@coinbase/cds-lottie-files'` for web if your project supports treeshaking
